# Красивый UI и полная админка
- Новый layout админ-панели с боковым меню и карточками.
- Страницы:
  - `/admin/accounts` — Ozon аккаунты (добавление, тест, синхронизация, удаление).
  - `/admin/reports` — выбор отчётов и каналов (дашборд/Telegram/E‑mail).
  - `/admin/settings` — чат Telegram, интервал автосинхронизации, SMTP.
- Все настройки выполняются из админ-панели. Бэкенд хранит глобальные настройки в таблице `setting`.
- После раскатки: `alembic upgrade head` применит миграцию `0005_settings`.
