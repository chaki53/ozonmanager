# Catalog & Preferences

Добавлены:
- `/reports/catalog` — список доступных отчётов;
- `/reports/preferences` GET/POST — сохранение видимости на дашборде и каналов (Telegram / Email) **для каждого пользователя**;
- `/reports/send` — выбор отчётов для выгрузки в TG и отправки на почту (PDF вложениями).

Фронт:
- `/dashboard` — переключатели видимости отчётов и каналов, а также простые виджеты для каждого отчёта.
В продакшене подключите реальные данные и auth-запросы к API.
